              # 1. 矩阵分解

              矩阵分解的作用：[1]

              • 矩阵填充（通过矩阵分解来填充原有矩阵，例如协同过滤的ALS算法就是填充原有矩阵）
              • 清理异常值与离群点
              • 降维、压缩
              • 个性化推荐
              • 间接的特征组合（计算特征间相似度）

              # 2. 特征分解

              # 2.1 特征值和特征向量

              如果一个向量 v\boldsymbol{v}v 是矩阵 A\boldsymbol{A}A 的特征向量，将一定可以表示成下面的形式：

              Av=λv \boldsymbol{A}\boldsymbol{v} = \lambda \boldsymbol{v} Av=λv

              其中，λ\lambdaλ 是特征向量 v\boldsymbol{v}v 对应的特征值，一个矩阵的一组特征向量是一组正交向量。

              # 2.2 特征值分解

              对于矩阵 A\boldsymbol{A}A，有一组特征向量 v\boldsymbol{v}v，将这组向量进行正交化单位化，就能得到一组正交单位向量。特征值分解，就是将矩阵 A\boldsymbol{A}A 分解为如下式：

              A=QΣQ−1 \boldsymbol{A} = \boldsymbol{Q}\boldsymbol{\Sigma}\boldsymbol{Q^{-1}} A=QΣQ−1

              其中，Q\boldsymbol{Q}Q 是矩阵 A\boldsymbol{A}A 的特征向量组成的矩阵，Σ\boldsymbol{\Sigma}Σ 则是一个对角阵，对角线上的元素就是特征值。

              解特征值的方式很简单，通过定义我们可以得到

              ∣A−λE∣=0 \left| \boldsymbol{A} - \lambda \boldsymbol{E} \right| = 0 ∣A−λE∣=0

              其中 E\boldsymbol{E}E 为单位矩阵。

              通过特征方程得到特征值，从而求出特征向量。

              # 3. SVD 分解

              # 3.1 奇异值分解

              奇异值分解是一个能适用于任意矩阵的一种分解的方法，对于任意矩阵 A\boldsymbol{A}A 总是存在一个奇异值分解：

              A=UΣVT \boldsymbol{A} = \boldsymbol{U}\boldsymbol{\Sigma}\boldsymbol{V}^{\mathsf{T}} A=UΣVT

              假设 A\boldsymbol{A}A 是一个 m×nm \times nm×n 的矩阵，那么得到的 U\boldsymbol{U}U 是一个 m×mm \times mm×m 方阵，U\boldsymbol{U}U 里面的正交向量被称为左奇异向量。Σ\boldsymbol{\Sigma}Σ 是一个 m×nm \times nm×n 的矩阵，Σ\boldsymbol{\Sigma}Σ 是对角矩阵，对角线上的元素称为奇异值。
